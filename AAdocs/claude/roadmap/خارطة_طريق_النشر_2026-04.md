### 🔴 Task 3: إنشاء OCI ARM A1 Instance
**تحتاج منك:** حساب Oracle فقط
**أساعدك في:** كل شيء بعد ذلك

**الخطوات (30 دقيقة):**
```
1. اذهب إلى: https://cloud.oracle.com/free
2. سجّل بحساب Gmail جديد مخصص للمدرسة
3. اختر منطقة: me-qatar-1
4. أنشئ Instance:
   Shape: VM.Standard.A1.Flex
   OCPU: 4  |  RAM: 24 GB  |  Storage: 200 GB
   OS: Ubuntu 22.04 LTS ARM64
5. Security List → افتح المنافذ: 22, 80, 443
6. احفظ ملف المفتاح (.pem)
7. أرسل لي: IP السيرفر
```

**إذا لم تتوفر A1 مجانية:**
- الخيار أ: حاول 2:00-5:00 ص بتوقيت قطر
- الخيار ب: 2× AMD Micro مجانية
- الخيار ج: A1.Flex مدفوع ($4.5/شهر فقط)

**→ عندما تُرسل IP السيرفر: سأعطيك الأمر الواحد لإعداد كل شيء**
