
# أرشيف مراجع تشغيل المدارس الحكومية في قطر (مبدئي)

**تاريخ الإنشاء:** 2026-03-20 06:38:50

يحتوي هذا الأرشيف على:

1) **qatar_school_ops_catalog.xlsx** — فهرس بالعناصر والروابط (مصنّف حسب القطاع والفئة).
2) **qatar_school_ops_links.csv** — قائمة مبسطة (العنصر، الرابط) لاستخدامها في السكربتات.
3) **download_all.ps1** — سكربت PowerShell لتنزيل كل الروابط التي تنتهي بامتدادات قابلة للتنزيل (PDF, DOCX, XLSX, ZIP…).

> **تنبيه**: بعض الروابط هي أخبار/صفحات بوابات دون مرفقات مباشرة. السكربت يتخطّاها تلقائيًا أو يحاول استكشاف الروابط داخل الصفحة إذا كانت مباشرة.

## استخدام السكربت (Windows PowerShell 7+)
```powershell
# داخل مجلد الأرشيف بعد فك الضغط
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
./download_all.ps1 -CsvPath ./qatar_school_ops_links.csv -OutDir ./downloads
```

## بنية مجلد التحميل
- `downloads/` ستحتوي الملفات التي تم تنزيلها حسب أسماء العناصر بشكل آمن.

## ألوان وهوية الشحانية
يمكن لاحقًا تطبيق تنسيقات ألوان الهوية (ماروني/أخضر فاتح/أزرق فاتح) على ملف Excel حسب الطلب.
