## **🔹 الفئة A — مخالفات بسيطة (خفيفة)**

*تعالج بالتنبيه الشفهي أو الكتابي، والمتابعة السلوكية الصفية.*

| رقم | المخالفة                   | أمثلة                              | الإجراء                          |
| --- | -------------------------- | ---------------------------------- | -------------------------------- |
| A1  | عدم الالتزام بالزي المدرسي | نقص في الزي، قبعة/إكسسوارات مخالفة | تنبيه شفهي → تنبيه كتابي         |
| A2  | تأخر بسيط عن الحصة         | دخول بعد الجرس دون مبرر            | تسجيل تأخر → متابعة              |
| A3  | عدم إحضار الأدوات/الكتب    | تكرار نسيان المواد                 | توجيه + إشعار لولي الأمر إن تكرر |
| A4  | إزعاج داخل الصف            | حديث جانبي/قذف أوراق/تشويش         | تنبيه + تدوين في سجل السلوك      |
| A5  | مخالفة بسيطة في الأنشطة    | عدم الالتزام بتعليمات المشرف       | تنبيه + منع مؤقت من نشاط جزئي    |
