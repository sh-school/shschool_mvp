## 1. السياق والمشكلة

### 1.1 الوضع الحالي
النظام الحالي **يدوي بالكامل** — المدير يضيف كل حصة واحدة تلو الأخرى يدوياً.
لا يوجد توليد تلقائي أو تحقق من القيود المرنة أو موازنة أحمال المعلمين.

### 1.2 التحدي الفريد
مدرسة الشحانية **مدرسة مجمّعة** (إعدادي G7-G9 + ثانوي G10-G12):
- **28 فصل** (6 مستويات × 4-5 شعب + فصول ESE)
- **~80 معلم** (معلمون + منسقون)
- **أنصبة مختلفة**: 34 حصة/أسبوع إعدادي vs 35 حصة/أسبوع ثانوي
- **مواد مختلفة**: العلوم (إعدادي) تنقسم إلى كيمياء + فيزياء + أحياء (ثانوي)
- **7 حصص يومياً** (الأحد-الأربعاء) + 6 حصص (الخميس)

### 1.3 الأهداف
1. **جدول تلقائي بالكامل** — المدير يضغط زر واحد فيتولد الجدول كاملاً
2. **عدالة التوزيع** — لا معلم يُثقل أكثر من غيره
3. **عدم التعارض** — ضمان صفر تعارضات
4. **مرونة التعديل** — تبديل حصص ذكي + غياب + تفريغ
5. **شفافية** — كل معلم يرى جدوله + إحصائيات عادلة

## 3. الاستراتيجية الخوارزمية

### 3.1 النهج: Constraint Satisfaction + Greedy Heuristic

اختيار هذا النهج لأنه:
- **بسيط في التنفيذ** (لا يحتاج مكتبات خارجية)
- **مضمون النتائج** (يضمن تحقيق القيود الصلبة)
- **سريع** (< 5 ثوانٍ لـ 28 فصل)
- **قابل للتعديل** بعد التوليد

### 3.2 القيود الصلبة (Hard Constraints — يجب تحقيقها 100%)

| # | القيد | الوصف |
|---|-------|-------|
| HC1 | عدم تعارض المعلم | معلم واحد لا يُدرّس فصلين في نفس الوقت |
| HC2 | عدم تعارض الفصل | فصل واحد لا يأخذ مادتين في نفس الوقت |
| HC3 | النصاب الأسبوعي | كل مادة تأخذ عدد حصصها المحدد بالضبط |
| HC4 | سعة اليوم | 7 حصص (أحد-أربعاء)، 6 حصص (خميس) |
| HC5 | تخصص المعلم | المعلم يُدرّس فقط مادته المعتمدة |
| HC6 | الموارد المشتركة | لا تعارض في المعامل/الملاعب |

### 3.3 القيود المرنة (Soft Constraints — تُحسَّن قدر الإمكان)

| # | القيد | الوزن | الوصف |
|---|-------|-------|-------|
| SC1 | تتابع الحصص | عالي (10) | لا أكثر من 3 حصص متتالية لنفس المعلم |
| SC2 | فراغات المعلم | عالي (8) | تقليل الحصص الفارغة بين حصص المعلم |
| SC3 | توزيع المادة | متوسط (6) | توزيع حصص المادة على أيام مختلفة (لا حصتين نفس اليوم إلا إذا لزم) |
| SC4 | موازنة الأحمال | متوسط (5) | عدد حصص يومي متقارب لكل معلم |
| SC5 | الحصص الأولى | منخفض (3) | المواد الأساسية (عربي/إنجليزي/رياضيات) في الحصص الأولى |
| SC6 | البدنية بعد الاستراحة | منخفض (2) | التربية البدنية بعد الاستراحة (ح4 أو ح5) |

### 3.4 خوارزمية التوليد (Greedy + Backtracking)

```
الخوارزمية:
1. بناء قائمة المهام (Tasks):
   لكل فصل × لكل مادة × عدد الحصص → task(فصل, مادة, معلم)

2. ترتيب المهام حسب الأصعب أولاً (Most Constrained First):
   - المواد ذات المعلم الوحيد أولاً
   - المواد ذات النصاب الأعلى ثانياً
   - الموارد المشتركة (معامل/ملاعب) ثالثاً

3. لكل مهمة:
   أ. حساب الخانات المتاحة (يوم × حصة) التي لا تنتهك أي قيد صلب
   ب. ترتيب الخانات حسب أقل انتهاك للقيود المرنة
   ج. اختيار أفضل خانة
   د. إذا لم توجد خانة → backtrack (تراجع خطوة)

4. بعد الانتهاء:
   - حساب نقاط الجودة (Quality Score)
   - عرض التقرير (انتهاكات مرنة + إحصائيات)

5. تحسين اختياري (Local Search):
   - محاولة تبديل حصص لتقليل انتهاكات SC
   - تكرار حتى لا يوجد تحسين
```

```python
class SubjectClassAssignment(models.Model):
    """ربط المادة بالفصل بالمعلم — أساس التوليد التلقائي"""
    school = ForeignKey(School)
    class_group = ForeignKey(ClassGroup)
    subject = ForeignKey(Subject)

    academic_year = CharField(max_length=9)
    requires_lab = BooleanField(default=False)  # يحتاج معمل؟
    preferred_periods = JSONField(default=list)  # حصص مفضلة [1,2,3]
    is_active = BooleanField(default=True)

    class Meta:
        unique_together = ('class_group', 'subject', 'academic_year')
```

## 7. معايير الجودة

### 7.1 مؤشرات الأداء (KPIs)

| المؤشر | الهدف | الحد الأدنى |
|--------|-------|-------------|
| تعارضات صلبة | 0 | 0 (إلزامي) |
| نقاط الجودة | ≥ 85/100 | ≥ 70/100 |
| أقصى حصص متتالية لمعلم | ≤ 3 | ≤ 4 |
| فراغات المعلم/يوم | ≤ 1 | ≤ 2 |
| انحراف الأحمال بين المعلمين | ≤ 2 حصة | ≤ 4 حصص |
| زمن التوليد | < 5 ثوانٍ | < 30 ثانية |
| نسبة تغطية الغياب التلقائي | ≥ 90% | ≥ 75% |

### 7.2 معايير العدالة

1. **الحمل الأسبوعي**: كل معلم يُدرّس نصابه المحدد (±1 حصة)
2. **التوزيع اليومي**: لا معلم يُدرّس أكثر من 5 حصص/يوم
3. **البدائل**: توزيع عادل لحصص البدائل (لا معلم يغطي أكثر من 3 بدائل/أسبوع)
4. **التفريغ**: كل معلم يستحق حصة تفريغ واحدة/أسبوع على الأقل
5. **الحصص الأولى**: لا معلم يبدأ 7:10 كل يوم

## 9. الخلاصة

هذا المشروع يحوّل نظام الجدولة من **يدوي بالكامل** إلى **ذكي مؤتمت** يراعي:
- **معايير وزارة التربية القطرية** (أنصبة + توقيتات)
- **خصوصية المدرسة المجمّعة** (إعدادي + ثانوي)
- **العدالة والشفافية** بين المعلمين
- **المرونة** في التعامل مع الغياب والتبديل
- **أفضل الممارسات العالمية** في خوارزميات الجدولة
