# تقييم أثر حماية البيانات (DPIA)

## Data Protection Impact Assessment

**المؤسسة:** مدرسة الشحانية الإعدادية الثانوية للبنين  
**النظام:** SchoolOS Qatar v3  
**تاريخ الإعداد:** 19 مارس 2026  
**المرجع القانوني:** قانون 13/2016 (PDPPL)  
**الحالة:** مسودة — يحتاج مراجعة DPO

---

## 1. وصف النظام

SchoolOS هو نظام إدارة مدرسية متكامل يعالج بيانات شخصية حساسة تشمل:
- بيانات الطلاب القاصرين (742 طالب)
- البيانات الصحية (حساسية، أمراض مزمنة، أدوية)
- البيانات السلوكية والتأديبية
- بيانات الحضور والغياب
- الدرجات والتقييمات الأكاديمية
- بيانات أولياء الأمور وبيانات التواصل

---

## 2. ضرورة المعالجة

| نوع البيانات           | الغرض                 | الأساس القانوني   |
|:---------------------- |:--------------------- |:------------------|
| بيانات الطالب          | إدارة التسجيل والأكاديمي | التزام قانوني     |
| البيانات الصحية        | سلامة الطالب           | مصلحة حيوية       |
| بيانات السلوك          | تطبيق اللوائح           | التزام قانوني     |
| بيانات الحضور          | متابعة الانضباط         | التزام قانوني     |
| بيانات ولي الأمر       | التواصل والموافقة       | موافقة صريحة      |

## 3. تقييم المخاطر

| الخطر                | الاحتمال | التأثير   | مستوى الخطر | الإجراء              |
|:---------------------|:---------|:----------|:------------|:---------------------|
| تسريب البيانات الصحية | منخفض    | عالٍ جداً | عالٍ        | تشفير Fernet ✅       |
| وصول غير مصرح         | منخفض    | عالٍ      | عالٍ        | RBAC + 2FA ✅         |
| اختراق الحساب         | منخفض    | عالٍ      | متوسط      | Rate Limiting ✅      |
| بيانات في Git         | تم       | حرج      | تم الإصلاح ✅ |                      |
| فقدان البيانات        | منخفض    | عالٍ      | متوسط      | Backup يومي ✅        |

---

## 4. التدابير الأمنية المطبقة

### تقنية:
- تشفير البيانات الصحية بـ Fernet ✅
- مصادقة ثنائية (2FA) للإداريين ✅
- نظام صلاحيات متعدد الأدوار (RBAC) ✅
- سجل مراجعة شامل (AuditLog) ✅
- Rate Limiting على صفحة الدخول ✅
- CSP Headers ✅
- NGINX مع SSL ✅
- نسخ احتياطي يومي مضغوط ✅

### تنظيمية:
- موافقة ولي الأمر إلزامية قبل الوصول ✅
- سياسة تغيير كلمة المرور عند أول دخول ✅
- تعيين DPO (قيد التنفيذ)
