## نظرة عامة

منصة إدارة مدرسية متكاملة لمدرسة الشحانية الإعدادية الثانوية للبنين — دولة قطر.
مبنية على Django مع 15 تطبيقاً (App) تغطي كل جوانب العملية التعليمية والإدارية، متوافقة مع قانون حماية البيانات الشخصية القطري (PDPPL).

| `CustomUser` | مستخدم مخصص مع دعم 2FA/TOTP |

## الامتثال — قانون حماية البيانات الشخصية (PDPPL)

| المتطلب | التنفيذ |
|---------|---------|
| تسجيل التدقيق (م.10) | `AuditLog` عبر Django Signals — كل عملية CRUD |
| الموافقة (م.5) | `ConsentRecord` + `ParentConsentMiddleware` |
| تشفير البيانات (م.9) | Fernet encryption في `core/models/_crypto.py` |
| حق المحو (م.18) | REST API `/api/v1/erasure/*` |
| إبلاغ الخروقات (م.16) | تطبيق `breach/` مع تتبع 72 ساعة (Celery) |
| تقييد الوصول (م.7) | `SchoolPermissionMiddleware` + Role-based access |
| مسؤول حماية البيانات | `DPO_EMAIL` في الإعدادات |

## التشغيل المحلي

```bash
# 1. البيئة
cp .env.example .env    # ضبط المتغيرات
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt

# 2. قاعدة البيانات
python manage.py migrate
python manage.py seed_all   # بيانات تجريبية

# 3. التشغيل
python manage.py runserver              # HTTP (port 8000)
daphne -p 8001 shschool.asgi:application  # WebSocket
celery -A shschool worker -l info       # مهام خلفية
celery -A shschool beat -l info         # مهام مجدولة

# 4. Tailwind (تطوير)
npx tailwindcss -i static/css/tailwind_input.css -o static/css/tailwind.min.css --watch
```
